=head1 NAME

libquvi - Overview of the library

=head1 DESCRIPTION

libquvi is a cross-platform library with C API for parsing ad0be
flash media properties. "quvi" is an acronym for "query video".

=head2 C API reference documentation

 <http://quvi.sf.net/doc/>

=head1 ENVIRONMENT

=head2 LIBQUVI_SCRIPTS_DIR

The library will additionally look in the colon-separated list of
directories specified by the LIBQUVI_SCRIPTS_DIR environment variable.

=head2 LIBQUVI_SHOW_DIR

When defined, the library will output the paths of the searched directories to
the stderr.

=head2 LIBQUVI_SHOW_SCRIPT

When defined, the library will output the file names of the found scripts to stderr.
The output will also indicate whether the script was accepted by the library.

=head2 LIBQUVI_VERBOSE_SCRIPTS

When defined, scripts using the 'quvi/debug' module messaging functions will output
to stderr.

=head2 Proxy

Refer to the libcurl C API, or C<curl(1)>, documentation for a complete list of the
supported environment variables. libquvi uses libcurl to access the Internet.
